WoodClub.MainMembers Class Reference

The home screen view. Displays a list of all member and allows the user to pick a member to view and edit their data. Also includes the main ToolBar to facilitate many other functions including reports, permissions, lockers, adding credits to multiple members, filtering the view and sending a member an email. Detailed Description

The home screen view. Displays a list of all member and allows the user to pick a member to view and edit their data. Also includes the main ToolBar to facilitate many other functions including reports, permissions, lockers, adding credits to multiple members, filtering the view and sending a member an email.
